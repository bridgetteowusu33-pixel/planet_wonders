# Cooking Fun V2 — Illustrated Asset Prompts

Reference prompts for AI-generating consistent cooking game illustrations.
All images must be **transparent PNG** unless noted. Target the "kawaii 3D" style
from the reference mockup: rounded forms, warm palette, child-friendly, no sharp edges.

---

## Global Base Style (prepend to every prompt)

```
3D kawaii-style illustration for a children's cooking game (ages 5-10).
Soft rounded forms, warm colours, playful proportions.
Pixar/clay-render aesthetic — smooth surfaces, subtle ambient occlusion, no harsh shadows.
Transparent background (PNG with alpha).
No text, no watermarks, no logos.
Child-safe, non-scary, storybook quality.
```
