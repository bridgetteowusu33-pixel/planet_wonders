Create a high-quality mobile app UI screen for a children's educational app called Planet Wonders.

The visual style must match an existing Planet Wonders home screen with:
- Soft pastel gradient background (mint, sky blue, light green)
- Rounded, colorful cards and buttons
- Friendly, playful, premium children's app look
- Clean spacing and clear hierarchy

**BRANDING (IMPORTANT)**
- App name: Planet Wonders
- Tagline: Color the World
- Font style: rounded, kid-friendly (similar to Nunito / Baloo)
- No clutter, no decorative text blocks

**SCREEN TYPE:** [CHOOSE ONE: Story / Coloring / Drawing / Gallery]

**TEXT RULES (VERY IMPORTANT)**

Use only real, readable English.
Do NOT invent words.
Do NOT misspell.
Keep text short and simple.

Allowed UI text:
- "Planet Wonders"
- "Color the World"
- "World Explorer"
- "Stories"
- "Coloring Pages"
- "Start Drawing"
- "Story"
- "Coloring"
- "Drawing"
- "Gallery"
- "Did you know?"
- "Next"
- "Back"
- "Done"

**STORY SCREEN (if selected)**
- Top bar: "USA . Story" or "Ghana . Story"
- Large illustrated scene (soft 3D, friendly)
- Rounded story text card below image
- Example story text (exact style): "This is Ava. She loves learning about the world."
- Audio icon visible (speaker icon)
- Navigation buttons: Back | Next

**COLORING SCREEN (if selected)**
- Top bar: "USA . Coloring" or "Ghana . Coloring"
- Large central coloring canvas
- Rounded tool icons: Brush, Fill, Eraser, Undo
- Color palette with large circular colors
- Bottom buttons: "Did you know?" and "Done"

**DRAWING SCREEN (if selected)**
- Title: "Drawing"
- Prompt text: "Draw anything!"
- Clean blank canvas
- Simple tool icons
- "Done" button

**GALLERY SCREEN (if selected)**
- Title: "Gallery"
- Grid of saved artwork thumbnails
- Rounded cards
- Friendly empty state if no artwork

**VISUAL STYLE RULES**
- Must visually match the Planet Wonders home screen
- Bright but calm colors
- Rounded corners everywhere
- Large touch-friendly buttons
- Soft shadows
- No harsh lines
- No dense text

The result should look like a real, production-ready kids app screen that clearly belongs to Planet Wonders.

**HOW TO USE THIS PROMPT**
- Change only the SCREEN TYPE line each time: Story / Coloring / Drawing / Gallery
- Keep everything else identical
- This guarantees visual consistency across the app
